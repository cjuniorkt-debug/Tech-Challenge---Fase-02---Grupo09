# Tech Challenge: Classificação da Qualidade de Vinhos

## 1. Sobre o Projeto

Este projeto tem como objetivo aplicar técnicas de Machine Learning para classificar vinhos de acordo com sua qualidade utilizando características físico-químicas.

O estudo contempla desde a análise exploratória dos dados até a comparação de modelos, otimização de hiperparâmetros e interpretação dos resultados, resultando na construção de uma solução capaz de apoiar a priorização de vinhos para avaliação e certificação.

## 4. Principais Resultados

* Comparação entre diferentes algoritmos de classificação.
* Seleção do Random Forest como modelo final.
* Identificação das variáveis mais relevantes para a qualidade do vinho.
* Classificação dos vinhos em categorias de priorização.
* Exportação dos resultados para Excel para facilitar a análise dos lotes.
